How to capture uncertainty?¶

Bayesian statistics¶

  • Flexible statistical framework that allows uncertainty quantification.
  • Uses probability distributions to reason about unknowns.
  • Not consider a single mean but rather assign every possible mean a probability.
  • Probability distribution: maps parameter values to probabilities.
  • Bayesian statistics updates probiability distributions after observing data.

  • "When I bet on horses, I never lose. Why? I bet on all the horses." Tom Haverford

def gen_plot(successes=(0, 100, 1), failures=(0, 100, 1)):
    alpha_prior=5
    beta_prior=5
    if successes == 0 and failures == 0:
        title = r'Prior probability distribution on $\theta$'
    else:
        title = r'Posterior probability distribution of $\theta$ after having seen data'
        
    alpha = alpha_prior + successes
    beta = beta_prior + failures
    x = np.linspace(0, 1, 201)
    fig = plt.figure(figsize=(8, 6))
    ax = fig.add_subplot(111, xlabel=r'Chance of success (profit or conversion)', 
                         ylabel=r'Probability',
                         title=title)
    ax.plot(x, sp.stats.beta(alpha, beta).pdf(x), linewidth=3.)
    ax.set_xticklabels([r'0\%', r'20\%', r'40\%', r'60\%', r'80\%', r'100\%']);
    
    return

def gen_plot(success=(1, 100), failure=(1, 100)):
    alpha = success
    beta = failure
    fig = plt.figure(figsize=(8, 6))
    x = np.linspace(0, 1, 100)
    ax = fig.add_subplot(111, xlabel='Mean (probability of beating the market)', 
                         ylabel='Belief', 
                         title=r'Probability distribution')
    ax.plot(x, sp.stats.beta(alpha, beta).pdf(x), linewidth=3.)
    ax.set_xticklabels(['0\%', '20\%', '40\%', '60\%', '80\%', '100\%']);

Bayes Formula¶

PyMC3¶

  • Probabilistic Programming framework written in Python.
  • Allows for construction of probabilistic models using intuitive syntax.
  • Written by: John Salvatier, Chris Fonnesbeck, Thomas Wiecki
  • Upcoming 3.0 release!

Some necessary formalism:¶

$\text{mean} \sim \text{Beta}(1, 1)$

$\text{data} \sim \text{Bernoulli}(\text{mean})$

  • $\text{mean}$ represents the unkown parameter we want to infer, but can't observe directly.
  • Use data to update belief about parameter.

Conclusions¶

  • Probabilistic Programming allows you to quantify uncertainty.
  • Blackbox inference algorithms allow estimation of complex models automatically.
  • PyMC3 puts advanced samplers at your fingertips.
  • Big Data: New class of samplers can work in parallel on subsets of the data.
